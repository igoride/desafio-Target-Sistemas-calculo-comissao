# calculo-comissao-csharp
Este projeto implementa um sistema simples para cálculo de comissão de vendedores.
A aplicação lê dados estruturados (JSON), processa as vendas realizadas e calcula a comissão individual de cada vendedor com base em regras pré-definidas.

Funcionalidades

Leitura de arquivo JSON com vendas.

Cálculo de comissão por vendedor.

Acumulação dos valores por vendedor utilizando Dictionary<string, double>.

Exibição dos resultados no console.
